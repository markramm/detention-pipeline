Methodology

This map visualizes signal convergence — the overlap of multiple independent indicators that a county is being targeted for ICE detention facility expansion. A single signal means little; multiple signal types in the same county indicate a pattern.

How the score works

Each county receives a heat score based on the number and type of signals detected. Signals are weighted by predictive value. Each signal type is capped to prevent one category from dominating.

A convergence bonus rewards diversity: +10 points per signal type beyond 2, and +15 more if 5 or more types are present.

The thesis

Detention consultants target counties with budget distress, cooperative sheriffs, and existing infrastructure. The pitch follows a playbook: sheriff conference recruitment, closed-session presentations, NDA-protected negotiations, then a commission vote framed as fiscal salvation.

This map surfaces the signals early — when multiple independent indicators converge on the same county, that county is likely being pitched. Early detection enables democratic response.

Limitations

This is an early-warning system, not a confirmed list. A high score means multiple independent signals converge — it does not mean a facility deal is confirmed. False positives are expected. Some counties with active pipeline work may score low if their signals haven't been captured yet.
